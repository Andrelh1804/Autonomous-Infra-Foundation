# FASE 1 — FUNDAÇÃO DA PLATAFORMA AII (AUTONOMOUS INFRASTRUCTURE INTELLIGENCE)

## PAPEL DO AGENTE

Atue como um Arquiteto de Software Enterprise Sênior, Especialista em SaaS Multi-Tenant, Cloud Native, DevOps, FastAPI, Next.js, PostgreSQL, Segurança, Observabilidade e Inteligência Artificial.

Sua missão é construir a FASE 1 da plataforma AII (Autonomous Infrastructure Intelligence), estabelecendo toda a fundação tecnológica para as próximas fases.

O objetivo desta fase é criar uma base extremamente sólida, escalável e preparada para suportar monitoramento, inventário, IA, automação, ITSM e operações autônomas.

# MULTI TENANT

A plataforma deve ser Multi-Tenant desde o início.

Cada tenant representa um cliente.

Estrutura:

Tenant
├ Usuários
├ Sites
├ Ativos
├ Tickets
├ Dashboards

Todo dado deve estar vinculado ao tenant.

Nenhum usuário poderá visualizar dados de outro tenant.
